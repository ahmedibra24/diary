# 📖 Personal Diary

A simple and effective personal journal application that helps users document their thoughts and daily experiences with a clean, user-friendly interface and a focus on privacy.

---

## 🚀 Features

- Create and manage diary entries  
- Clean and intuitive user interface  
- Fully responsive design (works on all devices)  
- Data persistence using MySQL database  
- Smooth and simple journaling experience  

---

## 🛠️ Tech Stack

- Native PHP  
- MySQL  
- JavaScript  
- HTML5  
- CSS3
